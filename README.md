### Base Docker Image

- node:7.5.0

<a name="usage"></a>
### Usage

<a name="build"></a>
#### Build

```
$ docker build -t hferentschik/hubot
```

<a name="run"></a>
#### Run

```
$ docker run -e HUBOT_IRC_PASSWORD=<password> -e  ENV REDISTOGO_URL=<redis-url> -t hferentschik/hubot
```
